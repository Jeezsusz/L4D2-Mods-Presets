# L4D2-Mods-Presets
Tired of changing your addons one by one? Ask no further, L4D2 Mods Presets will help you to change your addons to your own saved presets in no more than two clicks.

¿How to use it?

First, and most obvious, you must have L4D2 installed, the program will automatically locate your L4D2's folder and access its Addon's txt file.
Simply create a folder in your desktop (or anywhere you want), name it whatever you wish and drag the .exe file you downloaded here to the folder you created, then open the .exe file and it'll create two folders, one that will store your L4D2 Addon Presets and the other your backups for them.

Once you're all set up, you may see a lot of Workshop Addons enabled or disabled (depending on which ones you have enabled, once you open the .exe file it'll be already scanning your game Addons).

Now you may open the game, where you can make your own preset by simply clicking ''Save current as preset'', once it's done you can use it whenever you want (preferably better if you close the game to apply the changes, there will be a prompt whenever loading another preset, but you can also switch presets while in the game, although you must restart the campaign or world you're playing in so the new preset you switched to applies.

If you're installing addons in-game but notice the program isn't detecting new Addons, you'll have to close the game and hit the ''Rescan'' button, this will make your freshly installed Addons appear in the program and ready to go as another preset.
